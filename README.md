## 🛠️ Setup Guide

### Step 1 — Firebase Project Banao

1. [console.firebase.google.com](https://console.firebase.google.com) pe jao
2. "Add project" → project name likho → Create
3. **Authentication** enable karo:
   - Build → Authentication → Get Started → Email/Password → Enable
4. **Firestore Database** enable karo:
   - Build → Firestore Database → Create database → Start in test mode → asia-south1
5. **Web App** add karo:
   - Project Settings (gear icon) → Your apps → Web → Register app
   - Config copy karo (API Key, Project ID, etc.)

### Step 2 — Local Setup

```bash
# Clone / download project
cd business-os

# Dependencies install karo
npm install

# Dev server start karo
npm run dev
```

### Step 3 — Vercel pe Deploy karo (FREE)

1. [vercel.com](https://vercel.com) pe account banao (GitHub se login karo)
2. "New Project" → GitHub repo select karo
3. Framework: **Vite** auto-detect ho jayega
4. "Deploy" click karo — 2 minute mein live! ✅
